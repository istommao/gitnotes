# 简介

## 本地初始化

```bash
git init
```

`执行效果`

```shell
Initialized empty Git repository in yourpath/.git/
```

`添加一个文件到仓库中`

```bash
touch file
git add file
```

`添加所有文件`

```bash
git add *
```


```bash
git commit -m "Add file"
```

`查看提交记录`

```bash
git log
```

## 克隆远程仓库

```shell
git clone git@github.com:istommao/gitnotes.git
```
