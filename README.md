# gitnotes

**什么是Git**

> Git 是一个分布式版本控制软件，最初由林纳斯·托瓦兹（Linus Torvalds）创作，
> 于2005年以GPL发布。最初目的是为更好地管理Linux内核开发而设计。

## Git安装


### Linux 安装

**Centos Red Hat**

```bash
yum install git
```

**Debian或Ubuntu Linux**

```bash
apt-get install git
```

**源码安装**

- Git官网下载
- 解压
- ./config
- make
- make install

### Mac OS 安装

**homebrew 安装**

```bash
brew install git
```

### Window 安装

**Cygwin 模拟环境**
